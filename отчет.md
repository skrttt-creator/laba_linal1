Отчёт по лабораторной работе
Сравнение методов решения СЛАУ

1. Постановка задачи

Реализовать три алгоритма решения систем линейных алгебраических уравнений:
- метод Гаусса без выбора ведущего элемента
- метод Гаусса с частичным выбором по столбцу
- LU-разложение (без перестановок)

Провести сравнительный анализ:
- времени работы при разной размерности матриц
- точности на плохо обусловленных матрицах (матрица Гильберта)
- эффективности при решении систем с одинаковой матрицей и множеством правых частей


2. Описание алгоритмов

2.1 Метод Гаусса без выбора

Прямой ход: исключение переменных с использованием диагональных элементов в качестве ведущих. На каждом шаге k для всех i > k вычисляется множитель m = a[i][k] / a[k][k], затем из i-й строки вычитается k-я строка, умноженная на m. Обратный ход: последовательное вычисление неизвестных снизу вверх. Метод прост в реализации, но неустойчив для матриц с малыми ведущими элементами.

2.2 Метод Гаусса с частичным выбором

На каждом шаге k перед исключением выбирается максимальный по модулю элемент в столбце k среди строк от k до n-1. Если найденный элемент больше текущего a[k][k], соответствующие строки меняются местами. Затем выполняется стандартное исключение. Это повышает устойчивость метода, так как ведущий элемент не может быть слишком маленьким.

2.3 LU-разложение

Матрица A представляется в виде произведения L и U, где L — нижняя треугольная матрица с единичной диагональю, U — верхняя треугольная. Разложение выполняется по формулам:
- U[i][j] = A[i][j] - sum_{k=0}^{i-1} L[i][k] * U[k][j] для j >= i
- L[j][i] = (A[j][i] - sum_{k=0}^{i-1} L[j][k] * U[k][i]) / U[i][i] для j > i

Решение системы A*x = b сводится к двум шагам:
- L*y = b (прямая подстановка) — решается сверху вниз
- U*x = y (обратная подстановка) — решается снизу вверх

При множестве правых частей разложение выполняется один раз, а подстановки повторяются для каждой правой части.

5. Выводы

5.1 Производительность
LU-разложение немного быстрее метода Гаусса для одной системы и значительно эффективнее при множестве правых частей. При k=100 ускорение достигает почти 60 раз.

5.2 Точность
Выбор ведущего элемента критически важен для плохо обусловленных матриц. Без выбора метод Гаусса может полностью провалиться уже при n=15 на матрице Гильберта. С выбором точность остаётся приемлемой.

5.3 Преимущество LU
При многократном решении систем с одинаковой матрицей LU-разложение является оптимальным выбором, так как затратное разложение выполняется однократно.

5.4 Масштабирование
Все методы демонстрируют ожидаемую сложность O(n³), что подтверждено экспериментально. Время выполнения хорошо аппроксимируется кубической функцией от размера матрицы.
