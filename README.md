# Лабораторная работа 1
## Сравнение методов решения СЛАУ: метод Гаусса и LU-разложение

Этот проект реализует лабораторную работу по сравнению методов решения систем линейных алгебраических уравнений:
- метод Гаусса без выбора ведущего элемента;
- метод Гаусса с частичным выбором главного элемента по столбцу;
- LU-разложение без перестановок;
- прямую и обратную подстановки для треугольных систем.

Также программа запускает три серии экспериментов:
1. Сравнение времени решения одной системы для разных размеров матрицы.
2. Сравнение времени при нескольких правых частях для одной и той же матрицы.
3. Проверка точности на плохо обусловленной матрице Гильберта.
4. Проверка масштабирования O(n³).

## Структура проекта

- `lab_solver.cpp` — основной файл с реализацией всех методов и экспериментов.
- `README.md` — описание проекта и инструкция по запуску.
- `Отчёт.docx` — отчёт по лабораторной работе.

## Требования

- Компилятор с поддержкой C++17 (g++ 7+, clang 5+, MSVC 2017+)
- Стандартная библиотека

## Сборка и запуск

### Linux / macOS (g++ или clang++)


1. Создайте новый консольный проект

2. Добавьте файл lab_solver.cpp в проект

3.Соберите решение (Ctrl+Shift+B)

4.Запустите (Ctrl+F5)

Параметры экспериментов
По умолчанию используются:

- размеры для одной системы: 100, 200, 500, 1000

- размер для эксперимента с несколькими правыми частями: 500

- числа правых частей: 1, 10, 100

- размеры матрицы Гильберта: 5, 10, 15

- seed: 42 (для воспроизводимости)

Для изменения параметров нужно отредактировать соответствующие массивы в коде:

```bash


vector<int> sizes = {100, 200, 500, 1000}; в функции exp1()

const int n = 500; и vector<int> ks = {1, 10, 100}; в функции exp2()

vector<int> sizes = {5, 10, 15}; в функции exp3()
```

Тип генерируемых матриц
- Случайные матрицы с элементами из диапазона [-1, 1] (равномерное распределение)

- Матрица Гильберта для проверки точности (плохо обусловленная)

- Все алгоритмы реализованы вручную, готовые библиотечные функции (Eigen, LAPACK и т.п.) не используются.

Что выводит программа
В консоль выводятся таблицы:

ЭКСПЕРИМЕНТ 1 — времена решения одной системы для разных размеров;

ЭКСПЕРИМЕНТ 2 — времена для нескольких правых частей и ускорение;

ЭКСПЕРИМЕНТ 3 — невязки на матрице Гильберта;

Что включить в отчёт
1. Постановку задачи.

2. Краткое описание методов.

3. Описание структуры программы (основные функции и их назначение).

4. Таблицы результатов экспериментов (скопировать из консоли).

Выводы:

- сравнение времени работы методов;

- влияние выбора ведущего элемента на точность;

- преимущество LU-разложения при нескольких правых частях;


Замечания
Все алгоритмы реализованы вручную; сторонние библиотеки для решения СЛАУ не используются.

Для замеров времени применяется 
```bash
std::chrono::high_resolution_clock.
```
Для воспроизводимости используется фиксированный seed генератора случайных чисел.

Выбор ведущего элемента реализован только в методе Гаусса; LU-разложение выполняется без перестановок.
